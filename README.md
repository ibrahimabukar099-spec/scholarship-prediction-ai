# 🎓 Scholarship Eligibility Prediction System (AI)

Mashruucan waa nidaam ku shaqeeya **Artificial Intelligence (AI)** kaas oo loogu talagalay in lagu saadaaliyo haddii uu arday u qalmo deeq waxbarasho (Scholarship) iyo haddii kale.

## 🚀 Tiknoolajiyada la isticmaalay:
* **Backend:** Python (Flask/Express)
* **AI Model:** XGBoost (Extreme Gradient Boosting)
* **Database:** SQLite (`scholarship_data.db`)
* **Explainability:** SHAP/Radar Charts (si loo sharraxo go'aanka AI-da)

## 📁 Fayllada Muhiimka ah:
* `xgboost_scholarship_final.pkl`: Waa maskaxda AI-da ee tababaran.
* `scholarship_dataset.csv`: Xogta ardayda (2,000 records).
* `templates/`: Bogga hore iyo Interface-ka (HTML/CSS).
